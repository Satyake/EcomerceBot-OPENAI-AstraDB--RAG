# EcomerceBot-OPENAI-AstraDB--RAG

A Retrieval-Augmented Generation (RAG) system built using AstraDB and the Flipkart e-commerce review dataset.

## Features

- Efficiently processes Flipkart e-commerce reviews.
- Built using AstraDB for robust and scalable database management.
- Leverages OpenAI technologies to enhance text processing and data retrieval.
